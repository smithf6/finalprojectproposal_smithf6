Final Project Proposal

Interactive Walk-In Fridge

The idea I had for a website was to make a visual layout for a fridge that could be used to keep track of what items were in stock in a restaraunt setting.
The general layout would be a view of the fridge with every shelf visable, and a way to select between them. On each shelf would contain individual items
that can be selected to with an animation to designate whats being viewed. Items would show the number of protions left and alert for 0 left.
Another feature would be a prep list page with a way to enter values for the remaining portions. This would act as a managerial type interface that someone
could use to adjust the visuals of the interactive fridge.
